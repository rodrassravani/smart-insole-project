Smart Insole System 👟📊

An IoT-based wearable healthcare solution designed to monitor foot pressure distribution, gait patterns, motion, and temperature in real time using embedded sensors and cloud connectivity. The system helps in early detection of foot-related abnormalities, supports rehabilitation, improves mobility assessment, and enables remote health monitoring.

🚀 Project Overview

The Smart Insole System is a wearable device integrated inside a shoe insole that continuously tracks a user's walking behavior and foot health. The system combines pressure sensors, motion sensing, temperature monitoring, and wireless communication to provide intelligent real-time feedback through the Blynk IoT platform.

This project is especially useful for:

Elderly individuals
Diabetic patients
Rehabilitation monitoring
Sports and fitness analysis
Gait and posture analysis

⚙️ Working Principle
Sensors embedded inside the insole collect:
Foot pressure data
Motion and gait information
Temperature readings
ESP32 processes the sensor data in real time.
Data is transmitted wirelessly through Wi-Fi.
Blynk IoT dashboard displays:
Pressure distribution
Temperature values
Motion analysis
Real-time monitoring graphs
The system detects abnormalities such as:
Uneven weight distribution
Flat-foot trends
Abnormal plantar heat
Balance instability
